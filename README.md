# Simula-o-Hospitalar-com-Fila-Ordenada-em-C
O projeto consiste na implementação de uma fila de prioridade aplicada a uma simulação de atendimento hospitalar, onde pacientes são atendidos de acordo com o nível de prioridade. A fila é mantida de forma ordenada, garantindo que pacientes em estado mais crítico sejam atendidos antes dos demais, simulando o funcionamento de um sistema de triagem
